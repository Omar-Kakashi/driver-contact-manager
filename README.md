## How to Use

1. Open `index.html` in your web browser
2. Click "Choose Excel File" to upload your driver data
3. The Excel file should contain these columns:
   - Captain Code
   - Driver Id
   - Vehicle Id
   - Phone NO
   - hotel name
4. Click on any driver's name to view their details
5. Use the "Call" button to initiate a phone call
6. Use the "WhatsApp" button to send a WhatsApp message

## Files

- `index.html` - Main HTML structure
- `styles.css` - Mobile-responsive styling
- `script.js` - Excel parsing and contact management logic

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for loading the Excel library from CDN)

## Mobile Usage

This app is optimized for mobile devices. Simply open the app in your mobile browser, upload the Excel file, and tap on the buttons to call or message drivers directly.
